## Objetivo oficial do projeto

Criar uma calculadora simples com quatro operações básicas, usando Windows Forms e .NET Framework 4.8, com visual inspirado na calculadora padrão do Windows.

## Escopo funcional inicial

A calculadora deve conter apenas:

1. soma;
2. subtração;
3. multiplicação;
4. divisão.
