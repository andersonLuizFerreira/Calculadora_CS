# Requisitos da calculadora

## Escopo do primeiro incremento

Aplicacao desktop Windows Forms para Windows, implementada em C#/.NET.

## Operacoes obrigatorias

```text
- adicao
- subtracao
- multiplicacao
- divisao
- porcentagem
- raiz quadrada (sqr)
- memoria (M)
- memoria somar (M+)
- memoria subtrair (M-)
- limpar memoria (MC)
```

## Historico

A primeira versao deve possuir historico de calculos.

## Precisao numerica

Resultados devem considerar exibicao com ate 10 casas decimais.

## Pendencias

```text
- confirmar comportamento exato do botao M;
- confirmar se historico deve ser apenas visual durante a sessao ou persistido em arquivo/banco;
- confirmar alvo .NET recomendado: net9.0-windows;
- autorizar inicio de implementacao.
```
