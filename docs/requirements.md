# Requisitos — Calculadora_CS

## 1. Requisitos funcionais iniciais

A aplicação deve permitir:

- somar;
- subtrair;
- multiplicar;
- dividir;
- limpar operação;
- visualizar números e resultados no visor;
- exibir até 10 casas decimais.

## 2. Requisitos não funcionais

O projeto deve priorizar:

- simplicidade;
- clareza estrutural;
- facilidade de estudo;
- baixo acoplamento;
- manutenção simples;
- arquitetura enxuta.

## 3. Fora de escopo inicial

Não fazem parte do escopo inicial:

- calculadora científica;
- histórico de operações;
- memória;
- parser matemático avançado;
- gráficos;
- persistência;
- banco de dados;
- múltiplas janelas;
- temas avançados;
- plugins;
- sincronização;
- cloud.
