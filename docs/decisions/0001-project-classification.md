# ADR 0001 — Classificacao inicial do projeto

## Status

Aprovada em 2026-05-16.

## Decisao

O projeto Calculadora_CS sera conduzido inicialmente como uma aplicacao desktop simples para Windows, usando C#/.NET e Windows Forms.

## Autonomia local

```text
AI_AUTONOMY_LEVEL: baixa
```

A IA deve solicitar confirmacao humana para decisoes estruturais, alteracoes de escopo, arquitetura e inicio de implementacao.

## Consequencias

```text
- a engenharia deve ser proporcional a um projeto simples;
- a UI deve ser Windows Forms;
- a logica de calculo deve ser documentada antes da codificacao;
- codigo de produto ainda nao deve ser implementado nesta etapa.
```
